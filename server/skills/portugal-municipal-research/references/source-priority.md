# Source Priority

Use esta ordem de prioridade ao responder:

1. **Diário da República / atos normativos oficiais**
2. **Regulamentos municipais publicados e identificáveis**
3. **Páginas municipais oficiais que descrevem procedimentos em vigor**
4. **Portais oficiais de submissão / balcões online**
5. **Documentos operativos municipais** como NIPs, checklists, normas de instrução
6. **Conteúdo contextual municipal** que ajuda a orientar mas não basta sozinho para afirmar obrigatoriedade

## Regra prática

- Se uma conclusão depende de um regulamento, citar o regulamento ou a publicação oficial.
- Se depende do modo atual de submissão, citar a página operacional municipal ou o portal.
- Se só existir pista contextual, devolver `[SOURCE NEEDED]`.
