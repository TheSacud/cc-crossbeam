# Source Separation

Separar sempre as conclusões em três planos:

## 1. Fonte oficial normativa

Usar para:
- artigos de regulamento
- avisos/publicações oficiais
- regras vinculativas

## 2. Fonte oficial operacional

Usar para:
- instruções de submissão em vigor
- normas de instrução / NIPs
- páginas de balcão e taxas

## 3. Fonte municipal contextual

Usar para:
- resumos informativos
- páginas explicativas
- orientação preliminar que ainda precisa de base formal

## Regra

Uma exigência processual pode apoiar-se em fonte operacional municipal.
Uma exigência jurídica ou técnica vinculativa deve ter base normativa ou, se não tiver, ficar assinalada como `[SOURCE NEEDED]`.
