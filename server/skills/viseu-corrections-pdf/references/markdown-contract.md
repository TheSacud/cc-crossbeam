# Markdown Contract

O renderer deve assumir:

- `#` título principal
- `##` secções principais
- `###` subsecções
- parágrafos corridos
- listas com `-`
- listas numeradas com `1.`
- tokens como `[SOURCE NEEDED]` preservados literalmente

## Regra

O renderer pode normalizar layout, mas não pode alterar sentido, ordem material ou conteúdo jurídico/técnico do markdown.
