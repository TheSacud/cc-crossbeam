# Layout Spec

## Objetivo

Criar um PDF sóbrio, institucional e claramente técnico, adequado a resposta a notificação municipal.

## Regras de paginação

- A4
- margens amplas e regulares
- cabeçalhos e subtítulos com hierarquia clara
- evitar blocos muito densos
- listas devem manter indentação e legibilidade

## Cabeçalho

- `CrossBeam`
- `Resposta a notificação municipal`
- município e local, quando disponíveis

## Rodapé

- número da página
- referência curta ao documento
- timestamp de geração

## Aspeto

- sem cores agressivas
- sem watermark de “draft” ao estilo California
- ênfase visual leve para `[SOURCE NEEDED]`
