# RMUE Sources

Official source registry for Viseu municipal regulation:

| Source | Link / identifier | Notes |
|--------|-------------------|-------|
| `Aviso n.o 12538/2020` | Diario da Republica, 2.a serie, 27-08-2020 - https://files.dre.pt/2s/2020/08/167000000/0033500372.pdf | Primary source for the RMUE in force used by the structured reference files in this repo. |
| `Projeto de alteracao ao RMUE` | Municipio de Viseu PDF dated 2024-07-03 | Public-consultation text only; do not treat it as enacted without a later official act. |
| `Aviso n.o 15225/2024/2` | Diario da Republica, 2.a serie, 23-07-2024 | Opens public consultation for amendment of the RMUE; useful for change tracking, not as in-force wording. |

## Use notes

- Start from the published RMUE in force.
- Treat public-consultation material as proposed text, not enacted text, unless another official act confirms approval.
- If the run depends on exact wording, fetch and cite the article directly from the official PDF.
- The structured files in this folder are working notes derived from `Aviso n.o 12538/2020`; they do not replace the official act.
