# RMUE Structure

Primary source:
- `Aviso n.o 12538/2020`, Diario da Republica, 2.a serie, 27-08-2020
- PDF: https://files.dre.pt/2s/2020/08/167000000/0033500372.pdf

Use this file first to route a query to the correct RMUE block.

## Chapter map

| Chapter | Articles | Topic |
|---------|----------|-------|
| `CAPITULO I` | `1-4` | General provisions, object, definitions, pre-application attendance |
| `CAPITULO II` | `5-13` | Prior control procedures: licensing, prior communication, public discussion, caution, execution timing |
| `CAPITULO III` | `14-16` | Conservation duties and municipal intervention on degraded buildings |
| `CAPITULO IV` | `17-22` | Administrative instruction, document organization, process format |
| `CAPITULO V` | `23-48` | Urbanization and building rules, special cases, urbanistic conditions, parking |
| `CAPITULO VI` | `49-64` | Occupation of public space during works |
| `CAPITULO VII` | `65-85` | Other procedures: urgent documents, legalization, execution projects, telas finais, offences |
| `CAPITULO VIII` | `86-88` | Final and complementary provisions |

## Guardrails

- RMUE is municipality-specific. Check RJUE separately for national regime and hierarchy.
- For exact operative wording, return to the official PDF and cite the article.
- Do not treat the 2024 public-consultation amendment as enacted text unless a later official act confirms it.
