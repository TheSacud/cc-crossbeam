## Operational checklist v1

Use for any dossier that includes:
- legalization of existing works
- notice pointing to mismatch between built reality and licensed design
- follow-up after corrective / adaptation works

Minimum items already grounded in official sources:
- identification of the property and applicant legitimacy
- architectural survey of the existing building
- descriptive report framing the legalization request
- photographic survey of building and surroundings
- extracts from applicable municipal plans and constraints
- georeferenced implantation / location framing
- alteration drawings using the RMUE color logic
- execution specialty projects when corrective works are required
- `telas finais` whenever execution changes occurred

Items still depending on exact current NIP confirmation:

- exact inspection-request packaging, if any
